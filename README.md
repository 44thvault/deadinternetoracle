# DEAD INTERNET ORACLE

Digital necromancy via the Wayback Machine. Intercept real transmissions from the abandoned web.

## Deploy to Vercel

Push this folder to a GitHub repo and import it on [vercel.com/new](https://vercel.com/new). Zero config — Vercel auto-detects Next.js.

Or use the Vercel CLI:

```bash
npm install
npx vercel
```

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
